Linux的基本命令

ls 显示目录下的内容

-a：显示所有文件；
–color=when：支持颜色输出，when 的值默认是 always（总显示颜色），也可以是 never（从不显示颜色）和 auto（自动）；
-d：显示目录信息，而不是目录下的文件；
-h：人性化显示，按照我们习惯的单位显示文件大小；
-i：显示文件的 i 节点号；
-l：长格式显示；

cd 切换所在目录

mkdir 常见目录 文件夹 文件

rmdir 删除空目录

touch 修改文件的时间戳

千万不要把 touch 命令当成新建文件的命令，牢牢记住这是触摸的意思

-a：只修改文件的访问时间（Access Time)
-c：如果文件不存在，则不建立新文件
-d：把文件的时间改为指定的时间
-m：只修改文件的数据修改时间（Modify Time）

stat 显示文件或文件系统的详细信息

cat 连接文件并打印输出到标准输出设备

-A：相当于 -vET 选项的整合，用于列出所有隐藏符号；
-E：列出每行结尾的回车符 $；
-n：显示行号；
-T：把 Tab 键 ^I 显示出来；
-V：列出特殊字符；

rm 删除文件或目录

-f：强制删除（force）
-i：交互删除，在删除之前会询问用户
-r：递归删除，可以删除目录（recursive）

cp 复制文件和目录

mv 移动文件或改名
